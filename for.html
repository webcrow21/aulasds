Estrutura de Repetição for em Java

Hoje, vamos aprender a usar a estrutura for para repetir blocos de código com contagem controlada.

  • Estrutura: for.
  • Ideal para iterações com número fixo de repetições.
  • Combina com if e operadores lógicos!

Exemplo simples:

for (int i = 1; i <= 5; i++) {
    System.out.println("Contagem: " + i);
}

Estrutura for

Descrição

Repete um bloco por um número definido de vezes, usando inicialização, condição e incremento.

Sintaxe

for (inicialização; condição; incremento) {
    // Código
}

Exemplo em Java

for (int i = 0; i < 3; i++) {
    System.out.println("Iteração " + i);
}

Saída:
Iteração 0
Iteração 1
Iteração 2

Contexto do Dia a Dia

Regar 5 plantas, uma por vez, contando cada uma.

Use for quando souber o número de repetições!

for vs. while/do-while

Diferenças

  • for: Ideal para contagem fixa (ex.: 1 a 10).
  • while: Condição geral, pode não executar.
  • do-while: Executa pelo menos uma vez.

Exemplo Comparativo

int i = 1;
// for
for (i = 1; i <= 3; i++) {
    System.out.println(i);
}
// while
i = 1;
while (i <= 3) {
    System.out.println(i);
    i++;
}
Escolha for para iterações controladas!

Exemplos Práticos

Soma de Números

int soma = 0;
for (int i = 1; i <= 5; i++) {
    soma += i;
}
System.out.println("Soma: " + soma);

Verificação de Pares

for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
        System.out.println(i + " é par");
    }
}
Use for com if para decisões em laços!
