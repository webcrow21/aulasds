Estruturas de Repetição em Java

Hoje, vamos aprender a usar while e do-while para repetir blocos de código com base em condições.

  • Estruturas: while e do-while.
  • Usadas para automatizar tarefas repetitivas.
  • Combinam com if e operadores lógicos!

Exemplo simples:

int contador = 1;
while (contador <= 5) {
    System.out.println("Contagem: " + contador);
    contador++;
}
Vamos explorar cada estrutura nos próximos slides!

Estrutura while

Descrição

Repete um bloco enquanto a condição for true. Verifica a condição antes de executar.

Exemplo em Java

int saldo = 10;
while (saldo > 0) {
    System.out.println("Saldo: " + saldo);
    saldo--;
}

Saída:
Saldo: 10
Saldo: 9
...

Contexto do Dia a Dia

Continue regando as plantas enquanto o balde tiver água.

Use while para repetições com condição inicial!

Estrutura do-while

Descrição

Executa o bloco pelo menos uma vez e repete enquanto a condição for true.

Exemplo em Java

int tentativas = 0;
do {
    System.out.println("Tentativa " + (tentativas + 1));
    tentativas++;
} while (tentativas < 3);

Saída:
Tentativa 1
Tentativa 2
Tentativa 3

Contexto do Dia a Dia

Tente abrir a porta pelo menos uma vez, e continue se a chave não funcionar.

Use do-while quando precisa executar pelo menos uma vez!

while vs. do-while

Diferenças

  • while: Verifica a condição antes. Pode não executar.
  • do-while: Executa pelo menos uma vez, verifica depois.

Exemplo Comparativo

int x = 0;
// while
while (x > 0) {
    System.out.println("while: " + x);
}
// do-while
do {
    System.out.println("do-while: " + x);
} while (x > 0);

Saída:
do-while: 0

Escolha a estrutura com base na necessidade de execução inicial!
