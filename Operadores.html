Operadores Lógicos em Java

Hoje, vamos aprender como usar operadores lógicos para combinar condições em programas com if.

  • Operadores lógicos: && (E), || (OU), ! (NÃO).
  • Usados para tomar decisões mais complexas.
  • Vocês verão esses operadores nos exercícios de hoje!

Exemplo simples:

if (idade >= 18 && temIngresso == 's') {
    System.out.println("Entrada liberada!");
}
Vamos explorar cada operador nos próximos slides!

Operador && (E/AND)

Descrição

Retorna true se AMBAS as condições forem verdadeiras.

Exemplo em Java

int idade = 20;
char temIngresso = 's';
if (idade >= 18 && temIngresso == 's') {
    System.out.println("Entrada liberada!");
}

Saída: Entrada liberada!

Contexto do Dia a Dia

Precisa de dinheiro E carteira para comprar algo na loja.

Use && quando todas as condições devem ser atendidas!

Operador || (OU/OR)

Descrição

Retorna true se PELO MENOS UMA condição for verdadeira.

Exemplo em Java

double nota = 6.5;
int frequencia = 80;
if (nota >= 7 || frequencia >= 75) {
    System.out.println("Aprovado!");
}

Saída: Aprovado!

Contexto do Dia a Dia

Pode sair se for sábado OU domingo.

Use || quando basta uma condição ser verdadeira!

Operador ! (NÃO/NOT)

Descrição

Inverte o valor lógico: true vira false, e false vira true.

Exemplo em Java

boolean estaChovendo = false;
if (!estaChovendo) {
    System.out.println("Pode passear!");
}

Saída: Pode passear!

Contexto do Dia a Dia

Se NÃO está chovendo, leva o cachorro para passear.

Use ! para inverter uma condição!
